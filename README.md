# 🐱 胖猫的创意空间

一个由猫咪AI助手创建的前端浏览器页面，展示胖猫（小c）的技能和创意项目。

## 🌟 功能特性

### 🎨 视觉设计
- **响应式布局**：适配各种屏幕尺寸
- **猫咪主题**：可爱的猫咪元素和动画
- **动态主题**：支持实时切换主题颜色
- **平滑动画**：CSS和JavaScript动画效果

### 🛠️ 交互功能
- **点击计数器**：记录用户点击次数
- **项目演示**：三个创意项目的交互演示
- **主题切换**：5种预设颜色主题
- **时间显示**：实时显示当前时间
- **键盘快捷键**：支持键盘操作

### 📱 项目展示
1. **猫咪天气应用** - 用猫咪表情显示天气
2. **代码片段管理器** - 帮助管理常用代码
3. **任务追踪看板** - 可视化任务管理

## 🚀 快速开始

### 在线访问
访问GitHub Pages部署的页面：https://zanephelios-d.github.io/creat/

### 本地运行
1. 克隆仓库：
```bash
git clone https://github.com/zanephelios-d/creat.git
```

2. 打开 `index.html` 文件：
- 直接双击文件在浏览器中打开
- 或使用本地服务器：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .
```

3. 访问 `http://localhost:8000`

## 🎯 使用指南

### 主要功能
1. **主题切换**：点击颜色圆点切换页面主题
2. **猫咪计数器**：点击"点击我！"按钮增加计数
3. **项目演示**：点击"查看演示"按钮查看项目动画
4. **时间显示**：点击页脚链接显示当前时间

### 键盘快捷键
- **空格键**：增加点击计数
- **数字键1**：查看天气应用演示
- **数字键2**：查看代码管理器演示
- **数字键3**：查看任务看板演示
- **ESC键**：关闭演示窗口
